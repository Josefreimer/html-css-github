Para utilizar o peso da fonte, você deve primeiramente, colocar entre aspas simples: font-family:'work Sans', complemento,complemento; porém a 'work Sans' não existe em nenhum computador ou aparelho, para que ela funcione, você deve utilizar o seguinte comando dentro do style dentro de head: @inport url(); Após tudo isso, para definir qual será o estilo da fonte, você deverá utilizar o comando dentro de body: font-weight: Do menor ao maior Você pode Mudar o estilo junto com o peso da fonte, basta utilizar dentro de body o seguinte comando: font-style: italic; ou font-style:ctrl-space; E Você também poderá deixar o Texto sublinhado, basta utilizar dentro de body o seguinte comando: text-decoration: underline; Tudo isso, pode funcionar dentro de títulos, por exemplo: h1{ font-family: work Sans, Sans-serif; font-weight:bolder; font-size:2em ou 32px; font-style:italic; } porém tudo isso pode ser simplificado e é chamado de "Shorthand font" mas para que fique simplificado, você precisa de uma ordem para as coisas, que segue a seguinte ordem: font-style, font-weight,font-size,font-family Por exemplo: h1{font:italic bold 1.5em 'work Sans', Sans serif};
