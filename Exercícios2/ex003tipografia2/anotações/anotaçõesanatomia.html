Anatomia tipografia: Todas os tamanhos das fontes, tem as letras minúculas inspiradas no tamanho da letra "x" minúscula, porém, algumas letras acabam fugindo desse padrão, por exemplo a letra "g" percebe-se que ela tem a parte de baixo um pouco maior que a maioria das letras, e ela não é a unica, tem outras também. chamado de altura x A altura das maiúsculas, utiliza a altura x como base, e a parte de cima das letras que é maior que as letras minúsculas, é chamado de "altura das maiúsculas" Assim como nas minúsculas, algumas letras maiúsculas também fogem do padrão de tamanho, Quando foge para cima, se chama ascendente Quando foge para baixo, se chama descendente. A altura total da leta, juntando o ascendente e descendente, é chamado de "Corpo" Serifas: As serifas são o traços e prolongamentos que ocorrem no final das hastes, Lembrando que:Nem todas as fontes, têm serifas exemplos de Serifas: As serifas tem como importância, facilitar a leitura, Onde é por causa delas que conseguimos ler textos rapidamente e com facilidade, já que o cérebro não lê as letras, e sim as palavras em si, e as serifas ajudam a alinhar cada letra, o que ajuda na leitura Haste, são as partes a letra em que contém uma linha reta para cima, por exemplo a letra "K" logo na primeira parte da letra, dá para ver uma reta para cima, com duas retas para os lados um pouco inclinadas que também são hastes Tem também a haste deitada, porém, que obrigatoriamente precisa estar conectada com duas hastes.que é chamada de "filete" uma letra como exemplo é a letra "A" onde entre duas hastes, contém uma filete ligando Quando uma letra tem uma curva brusca, por exemplo o "n" essa curva é chamada de "arco" Quando a letra contém meio que um salto como apoio, como por exemplo a letra "y", é chamada de esporão. Quando uma letra tem uma volta reta por exemplo, A parte de baixo do "v" é chamado de "Vértice" Temos o "terminal", que é parecido com o filete, porém ele não liga mais de uma haste, como por exeplo a letra "r" onde tem uma haste e um terminal ligando a apenas uma haste Componentes corporais: Quando tem a parte que sai de uma haste para cima, porém que esteja ligando a uma continuação da letra, é chamada de "braço" Quando tem a parte que sai de uma haste para baixo, que esteja ligando a uma continuação de uma letra, é chamada de "perna" A base que esteja segurando a letra, é chamada de "pé" por exemplo o "A"ele tem duas bases, ou seja, dois pés Existente apenas na letra "S" é a chamada de "espinha". Que são as ligações das duas curvas do "S" A parte que liga as duas hastes é chamada de "barriga", por exemplo a letra "b", onde tem uma haste e outra parte da letra, que é a "Barriga" E dentro das letras que possuem uma aréa aberta, como por exemplo, a letra "O" onde existe buraco entre as curvas, é chamada de "olho" Em algumas fontes o existe uma parte lá em cima da letra "g" onde tem os dois olhos e logo ao lado do olho existe um traço, que é chamado de "orelha" E também na letra "g" existe a parte de baixo, o olho de baixo, que na verdade é chamado de cauda, pois, está ligando a parte de uma letra, outro exemplo é a letra "Q" Cada letra é chamada de "glifo" e quando pegandos todas as letras(glifos) é chamado de fonte Cada conjunto de letra, existe não somente uma família de letras, mas também existe diferentes formatos para cada letra, Que são chamados de família tipográfica, onde se destacam de letras das mais finas para as mais grossas, nessa ordem: light, normal, seminegrito, negrito, extranegrito Categorias das fontes: Serifadas: fontes que contém serifas sans-serif:Fontes que não contém serifas monoespaçada:sem serifa e monoespaçada handwriting:Tenta simular a escritura manual display: fontes que não contém nenhuma das características anteriores
